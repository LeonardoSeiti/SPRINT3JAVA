Bem vindo

Desenvolvi uma aplicação em Java usando Spring Boot com Thymeleaf, com o objetivo de apresentar a proposta do meu projeto para um futuro chatbot. Esta aplicação serve como uma demonstração inicial, ilustrando a estrutura e as funcionalidades que serão implementadas na versão final do chatbot.

A escolha do Spring Boot proporcionou uma base sólida e flexível, permitindo a criação de uma aplicação web eficiente e com uma estrutura bem organizada. A utilização do Thymeleaf para renderização de páginas dinâmicas foi essencial para criar uma interface amigável e interativa, facilitando a navegação e a compreensão do projeto.

Neste estágio, a aplicação permite uma visão inicial do potencial do chatbot, focando na experiência do usuário e na forma como as interações serão conduzidas. O projeto representa o primeiro passo no desenvolvimento de um assistente virtual capaz de realizar conversas automatizadas, com o objetivo de atender e auxiliar usuários em diversas tarefas.
